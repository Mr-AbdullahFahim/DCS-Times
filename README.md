# üìÖ UoJ DCS Timetable App

A native mobile application built for students of the **Department of Computer Science, University of Jaffna**. It provides a real-time, mobile-friendly interface to view daily lecture schedules and venue bookings directly from the official Google Sheets.

![License](https://img.shields.io/github/license/Mr-AbdullahFahim/DCS-Times)
![Version](https://img.shields.io/github/v/release/Mr-AbdullahFahim/DCS-Times)

## üöÄ Features

* **Real-time Sync:** Fetches the latest schedule updates instantly.
* **Smart Filtering:** Filter by Level (1, 2, 3, 4) and Day.
* **Venue Booking:** Dedicated tab for Auditorium and Lab bookings.
* **Auto-Update:** Built-in self-updater detects new GitHub Releases and prompts for installation.
* **Offline Support:** Caches data for viewing without an active connection.

## üì≤ Installation

### For Users (Android)
Download the latest `.apk` from the [Releases Page](https://github.com/Mr-AbdullahFahim/your-repo-name/releases).

### For Developers
1.  **Clone the repository**
    ```bash
    git clone https://github.com/Mr-AbdullahFahim/DCS-Times.git
    cd DCS-Times
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run locally (Web)**
    ```bash
    ionic serve
    ```

4.  **Run on Android**
    ```bash
    ionic cap open android
    ```
