# Security Policy

## Supported Versions

Only the latest major version of the app is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please **DO NOT** open a public issue.

Instead, please send an email to **fahimabdullah528@gmail.com**.

Please include:
* A description of the vulnerability.
* Steps to reproduce.
* Potential impact.

I will acknowledge your email within 48 hours and will keep you updated on the progress of the fix.
