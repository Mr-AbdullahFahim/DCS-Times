## ‚úàÔ∏è How to Contribute

1.  **Fork the Repository**: Click the "Fork" button at the top right of this page.
2.  **Clone your Fork**:
    ```bash
    git clone [https://github.com/YOUR_USERNAME/your-repo-name.git](https://github.com/YOUR_USERNAME/your-repo-name.git)
    ```
3.  **Create a Branch**:
    ```bash
    git checkout -b feature/AmazingFeature
    ```
4.  **Make Changes**: Write your code. Please follow the existing coding style (Prettier/ESLint).
5.  **Commit your Changes**:
    ```bash
    git commit -m 'Add some AmazingFeature'
    ```
6.  **Push to the Branch**:
    ```bash
    git push origin feature/AmazingFeature
    ```
7.  **Open a Pull Request**: Go to the original repository and click "New Pull Request".

## üß© Development Guidelines

* **TypeScript**: We use TypeScript. Please avoid using `any` types whenever possible.
* **Components**: Keep components small and reusable.
* **CSS**: Use the Ionic CSS utilities or `TabX.css` files for specific styling.
